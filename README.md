## Linked Lists

* A linked list is a data structure that represents a sequence of nodes.
* Time Complexity

|                 | Singly linked list | Doubly linked list | Array |
| --------------- | ------------------ | ------------------ | ----- |
| Search          | O(n)               | O(n)               | O(1)  |
| Add(at head)    | O(1)               | O(1)               | O(n)  |
| Remove(at head) | O(1)               | O(1)               | O(n)  |
